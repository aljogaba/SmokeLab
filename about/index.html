Manifiesto

Una bitácora viva

Smoke Lab es un proyecto personal para aprender barbecue mediante observación, documentación, repetición y análisis crítico.

Cada experimento registra lo que se hizo, lo que ocurrió y lo que debe cambiar. Las entradas públicas no buscan presentar perfección, sino compartir experiencia acumulada.

No buscamos una receta perfecta. Buscamos comprender qué decisiones producen cada resultado.

Principios del proyecto

Los procedimientos deben ser claros, las variables importantes deben registrarse y los errores deben conservarse como parte del aprendizaje.

Los protocolos no son definitivos: se actualizan cuando un experimento aporta evidencia práctica suficiente para cambiar una decisión.
