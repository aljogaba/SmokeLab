## Copiar a la carpeta local

1. Cierra temporalmente Live Server.
2. Haz una copia de seguridad de:

   `C:\GitHub\SmokeLab`

3. Extrae el ZIP.
4. Copia las carpetas `protocols` y `assets` dentro de:

   `C:\GitHub\SmokeLab`

5. Acepta reemplazar los archivos existentes.
6. Copia también `.nojekyll` a la raíz.
7. Reinicia Live Server.
